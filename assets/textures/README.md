# FFE Sample Textures

Reference textures for development and testing.

| File | Size | Description |
|------|------|-------------|
| white.png | 64x64 | Solid white — use as a placeholder sprite |
| checkerboard.png | 64x64 | Orange/blue checkerboard — useful for testing UV mapping |

These are RGB PNGs. loadTexture() decodes to RGBA8 internally (alpha = 255 for RGB sources).
