# Tutorials

Learn to build games with FastFreeEngine, step by step. Each tutorial builds a complete, playable game from scratch.

## 2D Games

- [Your First 2D Game](first-2d-game.md) -- Build a simple game with sprites, input, and collision detection
- More 2D tutorials coming soon (Pong, Platformer, Breakout)

## 3D Games

- [Your First 3D Game](first-3d-game.md) -- Load a 3D mesh, set up a camera, and add lighting
- More 3D tutorials coming soon (Materials, Skeletal Animation)

## Multiplayer

- [Multiplayer Basics](multiplayer-basics.md) -- Set up a local multiplayer game with client-server networking
- More multiplayer tutorials coming soon (Online Arena)

---

Each tutorial follows a consistent structure:

1. **What You Will Build** -- A description of the finished game
2. **Prerequisites** -- What you need to know before starting
3. **Step-by-step instructions** -- Code and explanations
4. **Complete Code** -- The full Lua listing at the bottom, ready to copy and run
