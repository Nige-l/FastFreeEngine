# Community

FastFreeEngine is built by people who love games and believe everyone deserves to make them. Here is how to get involved.

## GitHub

The engine is developed in the open on GitHub:

- [Repository](https://github.com/user/FastFreeEngine) -- Source code, issues, and pull requests
- [Contributing Guide](https://github.com/user/FastFreeEngine/blob/main/CONTRIBUTING.md) -- Code style, commit format, and PR process
- [Issue Tracker](https://github.com/user/FastFreeEngine/issues) -- Report bugs or request features

## License

FastFreeEngine is MIT licensed. Free and open source, forever. No royalties, no per-seat fees, no revenue share. See the [LICENSE](https://github.com/user/FastFreeEngine/blob/main/LICENSE) file for details.

## Discord

Coming soon. A Discord server for FFE developers, contributors, and students will be set up as the community grows.

## Code of Conduct

We are building a welcoming community for developers of all ages and backgrounds. A formal code of conduct will be published alongside the Discord server launch.
